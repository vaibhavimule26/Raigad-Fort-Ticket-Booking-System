# 🏰 Raigad Fort Smart Ticket Booking System

An AI-powered digital ticket booking and visitor management platform for **Raigad Fort**, designed to streamline ticket reservations, enhance security, and improve visitor experience through QR-based verification and role-based access control.

## 🚀 Features
- 🔐 OTP-Based User Authentication
- 🎫 Online Ticket Booking
- 📄 PDF Ticket Generation
- 📧 Email Notifications
- 📱 QR Code Generation and Validation
- 👥 Role-Based Access (User, Admin, Security)
- 📊 Admin Dashboard & Analytics
- 💬 AI Chatbot using Gemini API
- ⭐ Feedback Management
- 🛡 Secure JWT Authentication

## 🌍 Real-World Impact
Traditional ticketing systems often face long queues, manual verification, and poor visitor tracking. This project provides a smart digital solution by enabling:

- Paperless ticketing
- Secure visitor management
- QR-based entry validation
- Better administrative control
- Improved user experience
